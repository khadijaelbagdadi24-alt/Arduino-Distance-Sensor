# Arduino-Distance-Sensor

Distance Detection System with Visual and Audible Alerts.

### Hardware Components
* **Microcontroller:** Arduino Uno
* **Sensor:** HC-SR04 Ultrasonic Sensor
* **Visuals:** Multiple LEDs (Green, Yellow, Red)
* **Audio:** 5V Active Buzzer
* **Other:** Breadboard and Jumper Wires

### Key Features
* Real-time distance measurement using ultrasonic waves.
* Sequential LED alerts based on proximity.
* Audible alarm for critical safety distances.
